## Wie Tags funktionieren

Jede Seite kann mehrere Tags haben. Die Tags werden in den Metadaten der Seite definiert. Beispiel:

```markdown
---
title: Seite Titel
tags:
  - ki
  - coding
  - automatisierung
---
```

Diese Tags helfen bei der thematischen Suche und Navigation durch die Dokumentation.
